# ATM-Web-app

## Overview

The ATM Web Application simulates core ATM functionalities — including deposit and withdrawal — with secure login authentication. Users log in using their mobile number and PIN, manage their balance, and view recent transactions.

## Features

 User Authentication: Login using mobile number and PIN.

 Deposit Money: Add funds after login; updates balance.

 Withdraw Money: Withdraw amount securely with validation.

 Transaction History: Shows last transaction details (optional).

 Transaction Receipt: Displays or generates PDF receipt.

 Simple & Clean UI: Responsive and intuitive interface.

## Technologies Used

Frontend: HTML, CSS, JavaScript (Bootstrap UI)

Backend: Node.js / Python Flask / Spring Boot (depending on implementation)

Database: MySQL / MongoDB for storing user data and transactions

## Usage Flow

Login using mobile number and PIN.

Choose one of the following actions:

 Deposit: Enter amount → Confirm → Balance updated.

 Withdraw: Enter amount → Validate → Balance deducted.

View transaction summary.
